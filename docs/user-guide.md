# User Guide

## Project Library

Create a project for a novel, series, nonfiction book, screenplay, or other writing work. Projects are local by default.

## Structure

Break the work into parts, chapters, and scenes. Scenes can track goal, conflict, outcome, point of view, status, timeline placement, and target word count.

## Editor

The manuscript editor is for author text. AI output appears separately as suggestions and requires explicit review before it becomes part of project notes or accepted changes.

## AI Workshop

Configure a provider, choose context, and request help with structure, consistency, alternatives, and questions. AI can be disabled per project.

## Export

Export manuscript content to Markdown, HTML, or structured JSON in the first milestone.
